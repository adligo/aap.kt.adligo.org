# aap.kt.adligo.org
This will eventually contain base Kotlin-native code for native apps that want to embed a JVM in the native process.  
This is highly theortical, but I think it will work as I believe others have done simmilar things.   
https://github.com/adligo/aapi.kt.adligo.org/edit/main/README.md

Florian Weimer & Laurence Cable both suggested this approach on the JDK Discuss mailing list;
https://docs.oracle.com/javase/7/docs/technotes/guides/jni/spec/invocation.html
